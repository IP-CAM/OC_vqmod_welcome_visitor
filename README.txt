Welcome visitor

Shortens the text:
"Welcome visitor you can login or create an account"
to "Login or register"

Installation
=====================================================
Requires VQMOD.
Copy the vqmod directory to your store. That's all!


Usage
=========
Go to the front part.
Check how the welcome message looks like.
You can easily modify the text in the XML file provided in this extension.
